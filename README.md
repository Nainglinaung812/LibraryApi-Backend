# Library Management System (Web API)

This is a backend RESTful API built with **ASP.NET Core** and **EF Core** as part of my 10-day intensive backend learning challenge.

## 🚀 Features

- **CRUD Operations**: Manage books (Create, Read, Update, Delete).
- **Database**: Integrated with **SQL Server** using Entity Framework Core.
- **Containerization**: Fully dockerized environment for both the API and Database.
- **API Documentation**: Interactive UI with **Swagger/OpenAPI**.

## 🛠️ Tech Stack

- **Language**: C#
- **Framework**: .NET 9.0 (ASP.NET Core Web API)
- **ORM**: Entity Framework Core
- **Database**: Microsoft SQL Server
- **Tools**: Docker, Docker Desktop, VS Code

## 📦 How to Run with Docker

1. Clone the repository.
2. Run the following command in the terminal:
   ```bash
   docker-compose up --build
   ```
